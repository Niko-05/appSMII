Aplicación creada con la versión 3.10 de Python y Microsoft Visual Studio Code. Se utilizan librerías nativas de la versión de python aunque se usan dos que requieren de instalación.

Los requisitos adicionales para su funcionamiento son:
	
	- instalar OpenCV -> https://pypi.org/project/opencv-python/
	- instalar MediaPipe -> https://pypi.org/project/mediapipe/

Una vez instalados estos paquetes de forma correcta, simplemente haz doble click sobre el ejecutable o ejecutalo mediante orden de comandos.

Importante -> Solo funciona con la versión 3.10 de Python, si posees una versión más reciente has de instalar esta ya que MediaPipe solo es compatible hasta la versión mencionada (al menos a fecha de cuando este mensaje ha sido redactado).
